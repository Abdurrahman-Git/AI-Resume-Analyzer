# 📄 AI Resume Analyzer 🚀

An AI-powered web application that analyzes resumes and provides structured feedback using Large Language Models (LLMs).

---

## 🔥 Features

- 📤 Upload resume (PDF)
- 🧠 AI-powered analysis using LLM
- 📊 Score, strengths, weaknesses, suggestions
- 📥 Download analysis report
- ⚡ Clean and interactive UI

---

## 🛠 Tech Stack

- Python
- Streamlit
- OpenAI API
- PyPDF2

---

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
